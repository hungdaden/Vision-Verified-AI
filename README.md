# VisionVerify AI - Forensic Suite v3

**VisionVerify AI** là một ứng dụng pháp y hình ảnh chuyên sâu, sử dụng sự kết hợp giữa phân tích siêu dữ liệu (Metadata), phân tích cấu trúc pixel (Pixel-level analysis) và Deep Learning để xác định độ tin cậy của hình ảnh.

---

## 🚀 Chức năng chính
- **AI Generation Detection**: Xác định ảnh được tạo bởi các mô hình AI (DALL-E, Midjourney, Stable Diffusion).
- **Edit/Forgery Detection**: Phát hiện các dấu vết chỉnh sửa, cắt ghép, hoặc sử dụng filter (Lightroom/Instagram).
- **Deepfake Face Analysis**: Phân tích bề mặt da và kết cấu khuôn mặt để phát hiện mặt người giả lập.
- **Forensic Heatmap**: Hiển thị bản đồ nhiệt (Heatmap) các vùng có cấu trúc nén bất thường (ELA).
- **Full-screen UI**: Giao diện hiện đại, hỗ trợ kéo thả ảnh và hiển thị kết quả thời gian thực.

---

## 🧠 Các mô hình & Thuật toán phân tích

Hệ thống sử dụng chiến lược **Multi-Engine Fusion** (Kết hợp đa động cơ) để đưa ra kết quả chính xác nhất:

### 1. Phân tích AI (AI Detection)
- **EXIF Metadata Analysis**: Kiểm tra thông số Camera, ISO, tiêu cự. Ảnh AI thường thiếu EXIF hoặc có thông số không logic.
- **Deep Feature Analysis (ResNet-50)**: Sử dụng mạng thần kinh ResNet-50 để phân tích mật độ kích hoạt đặc trưng (Activation Density). Ảnh AI thường có mật độ kích hoạt khác biệt so với ảnh tự nhiên.
- **JPEG Ghost Detection**: Phát hiện dấu vết nén JPEG hai lần (Double Compression) - dấu hiệu thường thấy khi ảnh AI được lưu lại nhiều lần.
- **Format & Resolution Check**: Nhận diện các độ phân giải đặc trưng của mô hình AI (như 1024x1024) và định dạng file (PNG/WebP không EXIF).

### 2. Phân tích Chỉnh sửa (Edit Detection)
- **ELA (Error Level Analysis)**: Phân tích mức độ lỗi khi nén lại ảnh. Vùng bị chỉnh sửa sẽ có mức độ lỗi khác với phần còn lại.
- **Block Noise Inconsistency**: Chia ảnh thành các block 32x32 và đo độ nhiễu (noise). Ảnh ghép sẽ có sự chênh lệch nhiễu cực lớn giữa các vùng.
- **Sharpening/Filter Detection**: Phát hiện quầng sáng (halos) quanh biên cạnh - dấu hiệu của việc tăng sắc nét (Unsharp Mask) hoặc dùng filter.
- **Color Distribution**: Phân tích biểu đồ Histogram để phát hiện việc nén dải màu hoặc bão hòa màu quá mức (Color Grading).

### 3. Phân tích Deepfake
- **Laplacian Variance**: Đo độ sắc nét của kết cấu da mặt. Deepfake thường có bề mặt quá mịn (low variance) hoặc nhiễu kỹ thuật số (high variance) không tự nhiên.

## 🗺️ Composite Heatmap (Bản đồ nhiệt đa tầng)

Ứng dụng tạo ra một **bản đồ nhiệt đa tầng** kết hợp dữ liệu từ cả 3 mục phân tích. Heatmap tự động điều chỉnh hiển thị theo kết quả phát hiện chính:

| Layer | Colormap | Hiển thị khi | Ý nghĩa |
|-------|----------|-------------|----------|
| **ELA** | JET (xanh→vàng→đỏ) | Edited score cao nhất | Vùng đỏ/vàng = đã bị chỉnh sửa hoặc nén lại |
| **Noise** | HOT (đen→đỏ→vàng→trắng) | AI score cao nhất | Vùng sáng = noise bất thường do AI hoặc ghép ảnh |
| **Face** | Khung viền | Có khuôn mặt | 🟥 Đỏ = nghi deepfake · 🟩 Xanh = khuôn mặt bình thường |

**Cách hoạt động:**
- Khi **Edited** chiếm ưu thế → Layer ELA hiển thị 50%, Noise 10%
- Khi **AI Gen** chiếm ưu thế → Layer Noise hiển thị 50%, ELA 10%
- Khi **Deepfake** được phát hiện → Khung mặt đỏ với nhãn % được vẽ lên
- Khi **không phát hiện gì** → ELA nhẹ 30% để vẫn thấy cấu trúc ảnh

# VisionVerify AI - Forensic Suite v3

**VisionVerify AI** is an advanced image forensics application that combines Metadata analysis, Pixel-level structural analysis, and Deep Learning to verify image authenticity.

---

## 🚀 Key Features
- **AI Generation Detection**: Identify images generated by AI models (DALL-E, Midjourney, Stable Diffusion).
- **Edit/Forgery Detection**: Detect traces of editing, splicing, or filter usage (Lightroom/Instagram).
- **Deepfake Face Analysis**: Analyze skin texture and facial consistency to detect synthetic faces.
- **Forensic Heatmap**: Visualize areas with irregular compression structures (ELA).
- **Full-screen UI**: Modern interface with drag-and-drop support and real-time results.

---

## 🧠 Forensic Engines & Algorithms

The system uses a **Multi-Engine Fusion** strategy for maximum accuracy:

### 1. AI Detection
- **EXIF Metadata Analysis**: Verifies Camera, ISO, and focal length. AI images often lack EXIF or contain illogical parameters.
- **Deep Feature Analysis (ResNet-50)**: Uses ResNet-50 to analyze Activation Density. AI images show distinct activation patterns compared to natural photos.
- **JPEG Ghost Detection**: Detects traces of double JPEG compression—a common sign of AI images being re-saved.
- **Format & Resolution Check**: Recognizes AI-specific resolutions (e.g., 1024x1024) and formats (PNG/WebP without EXIF).

### 2. Edit Detection
- **ELA (Error Level Analysis)**: Analyzes compression error levels. Edited areas show different error rates than the original.
- **Block Noise Inconsistency**: Scans blocks for noise variations. Spliced images exhibit massive noise discrepancies between regions.
- **Sharpening/Filter Detection**: Detects halos around edges—a sign of Unsharp Masking or heavy filtering.
- **Color Distribution**: Analyzes Histograms to detect color compression or excessive saturation (Color Grading).

### 3. Deepfake Analysis
- **Laplacian Variance**: Measures facial texture sharpness. Deepfakes often have unnaturally smooth surfaces (low variance) or digital artifacts.

---

## 🗺️ Composite Heatmap (Multi-layer Visualization)

The application generates a **multi-layer composite heatmap** combining data from all 3 analysis categories. The heatmap automatically adjusts its display based on the dominant detection result:

| Layer | Colormap | Displayed When | Meaning |
|-------|----------|----------------|----------|
| **ELA** | JET (blue→yellow→red) | Edit score is highest | Red/yellow areas = edited or re-compressed |
| **Noise** | HOT (black→red→yellow→white) | AI score is highest | Bright areas = abnormal noise from AI or splicing |
| **Face** | Bounding boxes | Faces detected | 🟥 Red = suspected deepfake · 🟩 Green = normal face |

**How it works:**
- When **Edited** dominates → ELA layer at 50% opacity, Noise at 10%
- When **AI Gen** dominates → Noise layer at 50% opacity, ELA at 10%
- When **Deepfake** is detected → Red face bounding box with % label
- When **nothing detected** → Subtle 30% ELA overlay to show image structure

## 🛠 Installation

### System Requirements
- Python 3.10 or higher.
- NVIDIA GPU with CUDA support (optional, for faster ResNet analysis).

### Installation Steps
1. **Clone the repository**:
   ```bash
   git clone <https://github.com/hungdaden/Vision-Verified-AI>
   cd Image_Detect
   ```

2. **Create and activate a virtual environment**:
   ```bash
   python -m venv .venv
   # Windows:
   .\.venv\Scripts\activate
   # Linux/macOS:
   source .venv/bin/activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   python main.py
   ```

We provide an automated script to create the executable and desktop shortcut:
```bash
python build_app.py
```
*After completion:*
- The `.exe` file will be in the `dist/` directory.
- A shortcut will be automatically created on your **Desktop**.
